Umbra Tags — Manual

A desktop image library manager. Organize images with tags, search by tag hierarchy, and manage multiple libraries.

Libraries

A library is a folder on disk that Umbra Tags manages. Images imported into a library are copied into that folder and tracked in the save file. You can have up to 9 libraries.

ActionHow
Add a new libraryFile → Add New Library, or Ctrl+N
Open an existing library folderFile → Open Existing Library
Switch librariesAlt+1 – Alt+9, or the Libraries menu
Open library source folder in ExplorerCtrl+Enter

Importing Images

Images can be added to the active library by dragging files onto the gallery, or via the file dialog.

ActionHow
Import via file dialogCtrl+I
Import by drag and dropDrag image files onto the gallery
Delete source file after drag-inPreferences → Delete Source on Drag In
When importing, Umbra Tags computes a perceptual hash (dHash) for each image. If a visually similar image already exists in the library, a duplicate dialog appears with options to cancel, import anyway, replace the existing image, or delete the incoming file.

Searching

Type in the search box and press Enter. Searches match a single tag and include all child tags in the tag tree.

TermResult
allAll images in the library
untaggedImages with no tags, sorted by date modified
randimgOne random image from the library
any tag nameImages tagged with that tag, including child tags

Tagging

Tags are organized in a tree. A parent tag search includes all descendants.

ActionHow
Open tag editor for selectionSpace or Ctrl+T, or right-click → Edit Tags
Add a new tag to the treeTags → Add New Tag, or Ctrl+T from tag panel
Pin a tagRight-click a tag in the tag tree
